# AeroMerge
A Rust based TUI that pulls relevant information from across the aerospace and defense industry to give users a quick and easy updates, with cool features like a screensaver mode for when your at work!

## Features
- Intuitive TUI controls
- Cool animations
- Feeds categorized by type such as Defense, Space or Commercial
- Screensaver mode
